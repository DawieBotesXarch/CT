Overview

This documentation pack represents the complete Phase 1 design and specification work for the HappyDooda.co.za platform - a modern e-commerce solution for homeschooling digital resources, online courses, and educational content for the South African market.

Project Objectives

  • Migrate from WordPress/WooCommerce/TutorLMS to a modern, secure, and maintainable platform
  • Improve security and trust - Address WordPress security concerns
  • Enhance user experience - Better navigation, faster checkout, improved course platform
  • Full control - Complete ownership of codebase and infrastructure
  • Scalability - Built to grow with the business

Technology Stack

  • Frontend: Next.js 14+ (App Router), React 18+, Tailwind CSS, Shadcn/ui
  • Backend: Next.js API Routes, Prisma ORM, PostgreSQL
  • Authentication: NextAuth.js (JWT sessions)
  • Hosting: AWS Amplify (production), Docker (development)
  • Storage: AWS S3 + CloudFront CDN
  • Payments: PayFast (ZAR payment gateway)
